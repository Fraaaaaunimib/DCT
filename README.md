## Build
Innanzitutto scaricare il .NET 10 SDK da Microsoft.
Poi:
- `dotnet restore` per riscaricare tutti i pacchetti di dipendenze
- `dotnet compile` per compilarlo

## Esecuzione
Se il codice è stato compilato, eseguire `dotnet run` nella root della repository.

Altrimenti, si possono scaricare file eseguibili dalla tab Release di questa repository.
